# Playlist Maker

🎧 **Playlist Maker** — современное Android-приложение для поиска музыкальных треков, создания персональных плейлистов и управления списком избранного. Проект разработан в рамках учебного курса Яндекса с упором на чистую архитектуру, масштабируемость и актуальный стек технологий Android-разработки.
